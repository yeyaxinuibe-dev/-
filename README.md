# 个人介绍页面

单页个人介绍页，包含头像、履历、教育、兴趣爱好与联系方式，支持浅色/深色主题与响应式布局。

## 使用方式

1. 用浏览器直接打开 `index.html` 即可预览。
2. 将 `index.html` 中的占位内容替换为你自己的信息：
   - **头像**：把 `<img class="avatar" src="...">` 的 `src` 改为你的头像图片路径（本地路径或 URL）。
   - **姓名、职业、日期、地点**：在头部区块中修改。
   - **个人简介**：在「个人简介」区块中修改段落文字。
   - **工作经历 / 教育背景**：按现有结构复制或删减 `.timeline-item`，填写职位、公司、时间与描述。
   - **兴趣爱好**：修改说明文字，并增删 `.interest-card` 及图标。
   - **联系方式**：修改邮箱 `href` 与各社交链接的 `href`。
3. **下载简历**：在顶部「下载简历」按钮的 `href` 中填入你的简历 PDF 地址（如 `resume.pdf` 或完整 URL）；若暂不提供，可保留为 `#contact` 跳转到联系方式。

## 功能说明

- 浅色/深色主题切换（右上角按钮，会记住选择）
- 滚动时各区块淡入
- 「回到顶部」按钮（滚动一定距离后出现）
- 桌面与手机端自适应布局

## 技术说明

- 纯 HTML + CSS + 少量 JavaScript，无构建步骤。
- 字体：Noto Sans SC、Outfit（通过 Google Fonts 加载）。
- 当前头像为占位图（DiceBear），替换为你的图片即可。
